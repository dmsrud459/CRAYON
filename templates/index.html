Teachable Machine is a web-based tool that makes creating machine
learning models fast, easy, and accessible to everyone

Works with...

The models you make with Teachable Machine are real
TensorFlow.js models that work anywhere javascript
runs, so they play nice with tools like Flask, JS, jQuery, Node.js, Python, Visualstudio, bootstrap & more.

Plus, export to different formats to use your models elsewhere, like Coral, Arduino & more.
